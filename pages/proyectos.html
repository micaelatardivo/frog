CUARTO SET

Cuarto Set es un emprendimiento que se dedica a vender articulos de padel tales como paletas importadas y nacionales, pelotas, protectores, bolsos paleteros, etc. Con la necesidad de llegar a un público mayor y expandirse a otras ciudades Cuarto Set forma parte de nuestras ideas hace mas de dos años, con grandes avances Cuarto Set actualmente es uno de nuestros exitos en FROG.
